#Instalndo

Instale o **Node 6.0**  
Instale o **MongoDB 3+**  

Clone esse repositorio  
```sh
  git clone https://github.com/ITServDev/syncro-webservice.git
```

Instale as dependencias  
```sh
  npm install
```

### Comandos
```sh
  #Inicia o Server
  npm run serve

  #Inicia o Server em modo Debug
  npm run serve:debug
```
